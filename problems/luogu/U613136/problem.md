## 题目描述

给你一个整数数组 prices,其中prices[i]表示某支股票第 i 天的价格。

在每一天，你可以决定是否购买和/或出售股票。你在任何时候**最多**只能持有**一股**股票。然而，你可以在**同一天**多次买卖该股票，但要确保你持有的股票不超过一股。

返回你能获得的***最大***利润。

## 输入输出样例 #1

### 输入 #1

```
6
7 1 5 3 6 4
```

### 输出 #1

```
7
```

## 输入输出样例 #2

### 输入 #2

```
5
1 2 3 4 5
```

### 输出 #2

```
4
```

## 输入输出样例 #3

### 输入 #3

```
5
7 6 4 3 1
```

### 输出 #3

```
0
```

## 说明/提示

1<= prices.length <= 3×104

0<=prices[i]<=104
