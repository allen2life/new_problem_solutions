# P1981 [NOIP 2013 普及组] 表达式求值
## 题目背景

NOIP2013 普及组 T2

## 题目描述

给定一个只包含加法和乘法的算术表达式，请你编程计算表达式的值。

## 输入输出样例 #1

### 输入 #1

```
1+1*3+4
```

### 输出 #1

```
8
```

## 输入输出样例 #2

### 输入 #2

```
1+1234567890*1
```

### 输出 #2

```
7891
```

## 输入输出样例 #3

### 输入 #3

```
1+1000000003*1
```

### 输出 #3

```
4
```

## 说明/提示

对于 $30\%$ 的数据，$0≤$ 表达式中加法运算符和乘法运算符的总数 $≤100$。

对于 $80\%$ 的数据，$0≤$ 表达式中加法运算符和乘法运算符的总数 $≤1000$。

对于 $100\%$ 的数据，$0≤$ 表达式中加法运算符和乘法运算符的总数 $≤100000$。
