# P1450 [HAOI2008] 硬币购物
## 题目描述

共有 $4$ 种硬币。面值分别为 $c_1,c_2,c_3,c_4$。

某人去商店买东西，去了 $n$ 次，对于每次购买，他带了 $d_i$ 枚 $i$ 种硬币，想购买 $s$ 的价值的东西。请问每次有多少种付款方法。

## 输入输出样例 #1

### 输入 #1

```
1 2 5 10 2
3 2 3 1 10
1000 2 2 2 900
```

### 输出 #1

```
4
27
```

## 说明/提示

#### 数据规模与约定

* 对于 $100\%$ 的数据，保证 $1 \leq c_i, d_i, s \leq 10^5$，$1 \leq n \leq 1000$。
