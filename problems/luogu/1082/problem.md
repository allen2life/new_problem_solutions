# P1082 [NOIP 2012 提高组] 同余方程
## 题目描述

求关于 $ x$ 的同余方程 $ a x \equiv 1 \pmod {b}$ 的最小正整数解。

## 输入输出样例 #1

### 输入 #1

```
3 10
```

### 输出 #1

```
7
```

## 说明/提示

### 数据规模与约定

- 对于 $40\%$ 的数据，$2 \le b\le  10^3$；
- 对于 $60\%$ 的数据，$2 \le b\le  5\times 10^7$；
- 对于 $100\%$ 的数据，$2 \le a, b\le  2\times 10^9$。
