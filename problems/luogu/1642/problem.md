# P1642 规划
## 题目描述

某地方有 $N$ 个工厂，有 $N-1$ 条路连接它们，且它们两两都可达。每个工厂都有一个产量值和一个污染值。现在工厂要进行规划，拆除其中的 $M$ 个工厂，使得剩下的工厂依然连成一片且总产量 / 总污染的值最大。

## 输入输出样例 #1

### 输入 #1

```
3 2
2 3 4
1 1 1
1 2
2 3
```

### 输出 #1

```
4.0
```

## 说明/提示

### 数据范围及约定

对于全部数据，$1<N<100$，$1 \le M<N$，$1\le w_i\le 10000$，$1\le c_i\le 10000$。
