### 题意

每个式子由若干个 `s` 和 `c` 相加组成，其中：

- `s = sin^2(pi/7)`
- `c = cos^2(pi/7)`

可以从给定式子中选一些相加，要求结果是整数，并且这个整数尽量大。

### 思路

先看小数据暴力：

@include-code(./brute.cpp, cpp)

设：

- `a = sin^2(pi/7)`
- `b = cos^2(pi/7)`

有恒等式：

`a + b = 1`

于是一个式子只需要关心两件事：

1. 它一共有多少个字符，记作 `len`
2. 它的 `s` 比 `c` 多多少，记作 `diff = #s - #c`

如果最终选中的所有式子总共有：

- `S` 个 `s`
- `C` 个 `c`

那么总和是：

`S * a + C * b`

只有当 `S = C` 时，才能全部配成若干个 `a+b=1`，结果才是整数。

而这时答案正好就是：

`(S + C) / 2`

所以题目变成：

> 选出若干个式子，使总差值 `diff` 为 `0`，并让总长度最大。

这就是一个差值背包：

- 状态：总差值为某个值时的最大总长度
- 转移：每个式子选或不选

最后输出：

`dp[0] / 2`

### 复杂度

时间复杂度 `O(nm)`，空间复杂度 `O(m)`，其中 `m` 是题目中的总字符规模量。

### 总结

这题最关键的不是三角函数，而是把它转化成：

- 差值是否能配平
- 配平后总长度尽量大

一旦想到 `sin^2x + cos^2x = 1`，就会变成很自然的背包题。
