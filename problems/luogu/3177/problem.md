# P3177 [HAOI2015] 树上染色
## 题目描述

有一棵点数为 $n$ 的树，树边有边权。给你一个在 $0 \sim n$ 之内的正整数 $k$ ，你要在这棵树中选择 $k$ 个点，将其染成黑色，并将其他的 $n-k$ 个点染成白色。将所有点染色后，你会获得黑点两两之间的距离加上白点两两之间的距离的和的收益。问收益最大值是多少。

## 输入输出样例 #1

### 输入 #1

```
3 1
1 2 1
1 3 2
```

### 输出 #1

```
3
```

## 说明/提示

对于 $100\%$ 的数据，$0 \leq k \leq n \leq 2000$，$1 \leq w \leq 10^4$。
