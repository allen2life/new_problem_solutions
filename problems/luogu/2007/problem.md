# P2007 魔方

## 题目描述

给定一个 $3 \times 3 \times 3$ 三阶魔方当前六个面的颜色，以及一串只包含 `1`、`2`、`3`、`4` 的操作序列。每个操作表示题目图示中的一种转动。求按顺序执行所有操作后，魔方六个面的状态。

输入六个面的顺序固定为：前、后、左、右、上、下。每个面用三行、每行三个数字表示。

## 输入格式

第一行是操作序列，长度不超过 $100$。

接下来 $18$ 行，每 $3$ 行表示一个面，六个面依次为前、后、左、右、上、下。

## 输出格式

输出 $18$ 行，格式与输入的六个面相同，表示操作后的魔方状态。
