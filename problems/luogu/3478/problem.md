# P3478 [POI 2008] STA-Station
## 题目描述

给定一个 $n$ 个点的树，请求出一个结点，使得以这个结点为根时，所有结点的深度之和最大。

一个结点的深度之定义为该节点到根的简单路径上边的数量。

## 输入输出样例 #1

### 输入 #1

```
8
1 4
5 6
4 5
6 7
6 8
2 4
3 4
```

### 输出 #1

```
7
```

## 说明/提示

#### 样例 1 解释

输出 $7$ 和 $8$ 都是正确答案。

#### 数据规模与约定

对于全部的测试点，保证 $1 \leq n \leq 10^6$，$1 \leq u, v \leq n$，给出的是一棵树。
