# B3642 二叉树的遍历
## 题目描述

有一个 $n(n \le 10^6)$ 个结点的二叉树。给出每个结点的两个子结点编号（均不超过 $n$），建立一棵二叉树（根节点的编号为 $1$），如果是叶子结点，则输入 `0 0`。

建好树这棵二叉树之后，依次求出它的前序、中序、后序列遍历。

## 输入输出样例 #1

### 输入 #1

```
7
2 7
4 0
0 0
0 3
0 0
0 5
6 0
```

### 输出 #1

```
1 2 4 3 7 6 5
4 3 2 1 6 5 7
3 4 2 5 6 7 1
```

## 说明/提示

对于 $1\leq n\leq 10^5$ 的测试数据，时限为 1 秒。

对于 $10^5< n\leq 10^6$ 的测试数据，时限为 3 秒。
