### 题意

给出若干个城镇和已经修好的道路，问最少还要再修多少条路，才能让任意两个城镇之间都可以互相到达。

输入有多组数据，以单独一个 `0` 结束。

### 思路

先看一个小数据暴力：

@include-code(./brute.cpp, cpp)

暴力把城镇和道路建成无向图，然后 DFS 数连通块个数。

如果当前有 `k` 个连通块，那么最少只要再修 `k-1` 条路，就能把它们全部连成一个整体：

- 每修一条连接不同连通块的路，连通块数量就减少 `1`
- 从 `k` 个连通块变成 `1` 个，需要减少 `k-1` 次

正式做法可以直接用并查集维护连通块。

处理每组数据时：

1. 初始化并查集，每个城镇单独成集
2. 读入每条路 `(x, y)`，把两个城镇合并
3. 最后统计有多少个根节点，也就是有多少个连通块
4. 输出 `连通块数 - 1`

### 复杂度

设一组数据有 `n` 个点、`m` 条边。

并查集总复杂度为：

`O((n + m)\alpha(n))`

空间复杂度 `O(n)`。

### 总结

这题的本质就是“无向图有几个连通块”。一旦看出来答案只和连通块数量有关，写成 DFS 或并查集都很自然；这里用并查集会更短。
