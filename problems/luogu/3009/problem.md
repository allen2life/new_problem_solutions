# P3009 [USACO11JAN] Profits S
## 题目描述

奶牛们开始了新的生意，它们的主人约翰想知道它们到底能做得多好。这笔生意已经做了 $N(1\le N\le 10^5)$ 天，每天奶牛们都会记录下这一天的利润 $P_i(\vert P_i\vert ≤1,000)$。

约翰想要找到奶牛们在连续的时间期间所获得的最大的总利润。（注：连续时间的周期长度范围从第一天到第 $N$ 天）。

请你写一个计算最大利润的程序来帮助他。

## 输入输出样例 #1

### 输入 #1

```
7 
-3 
4 
9 
-2 
-5 
8 
-3
```

### 输出 #1

```
14
```

## 说明/提示

**【样例解释】**

取区间 $[2,6]$ 的和可以得到最大值 $14$。
