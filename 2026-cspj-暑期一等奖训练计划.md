# 2026 CSP-J 暑期一等奖训练计划

## 训练目标

本计划面向已经完成 `problem-sets/csp-j-400.md` 大部分题目的学生。当前主要问题不是题量不足，而是：

1. 能写出暴力解，但无法稳定超过 60 分。
2. 不知道如何从暴力解升级到正解。
3. 想到优化方向后，不能稳定证明正确性。
4. 知道正解思路后，代码实现和调试不够稳定。

因此暑假训练目标不是继续盲目刷题，而是把已经具备的基础题量转化成稳定考试分数。

目标分数结构：

```text
T1：100 分，必须稳定
T2：90-100 分，基本零失误
T3：70-100 分，重点突破
T4：30-60 分，稳定拿部分分
总分目标：290-330+
```

稳一等奖的核心不是依赖 T4 超常发挥，而是：

```text
T1/T2 不崩
T3 尽量打穿
T4 稳定拿暴力分、特殊性质分、部分正解分
```

## 核心训练方法

每一道核心题都按下面的链路训练：

```text
暴力解 -> 复杂度瓶颈 -> 优化方向 -> 正确性证明 -> 正解代码 -> 对拍/复盘
```

每题必须留下四行复盘：

```text
暴力怎么写：
瓶颈在哪里：
正解消掉了哪一层复杂度：
下次看到什么特征要想到这个方法：
```

如果只 AC 但写不出这四行，这道题不算真正过关。

## 每日训练结构

每日训练时间按 3-4 小时设计。

```text
60-90 分钟：限时做 1 道核心题
30 分钟：补全暴力解，确认暴力分能拿到
60 分钟：从暴力升级到正解
30 分钟：重写代码、调试或对拍
10 分钟：记录四行复盘
```

训练时不要只问“这题 AC 了吗”，而要问：

```text
暴力哪一层循环太慢？
数据范围暗示了什么复杂度？
能不能用 DP、前缀和、排序、贪心、二分、状态建模消掉瓶颈？
为什么优化后的做法一定正确？
代码能否在 40-60 分钟内稳定写出？
```

## 第 1-2 周：诊断与 DP 基础转化

目标：把“我只能写暴力”变成“我知道暴力哪里慢，并能尝试设计状态”。

训练重点：

1. 状态含义必须说清楚。
2. 转移来源必须能解释。
3. 初始化和边界不能靠猜。
4. 不能只背模板，要能从暴力枚举中抽出状态。


## 第 3-4 周：数学、贪心与规律专项

目标：解决“能写暴力，但不知道怎么优化”和“想到贪心但不会证明”。

训练重点：

1. 从样例和小数据中找规律。
2. 从数据范围反推需要的复杂度。
3. 用交换、局部最优、前缀最小值、排序后扫描等方式证明贪心。
4. 把数学推导写成可实现的程序。

## 第 7-8 周：模拟赛与稳分策略

目标：不再大规模扩题，集中训练考试分数转化。

每周安排：

```text
2 次完整模拟赛
2 次错题重写
1 次 T3/T4 专项
1 次 T1/T2 零失误训练
1 天轻量复盘
```

每次模拟赛后必须完成：

```text
1. 统计每题实际得分。
2. 标记失分原因：不会想、想到了但不会证、会想但写错、时间不够、低级错误。
3. T1/T2 的错误必须当天重写。
4. T3/T4 至少补出暴力和正解思路。
5. 第二天重写最关键的 1-2 道题。
```

考试时间分配建议：

```text
0-40 分钟：T1 完成并检查
40-100 分钟：T2 完成并检查
100-190 分钟：主攻 T3
190-240 分钟：T4 暴力分 + 特殊性质
最后 30 分钟：回查 T1/T2/T3
```

如果 T3 卡住超过 25 分钟还没有进展，必须先写暴力和部分分，不能把整场考试赌在一个想法上。

## 最终执行原则

1. 每天宁可少做题，也要完成“暴力到正解”的完整链路。
2. T1/T2 错误必须当天清零，不能积压。
3. T3 是一等奖核心突破点，复盘质量比题量更重要。
4. T4 不追求每题正解，但必须训练稳定拿部分分。
5. 每周模拟赛的分数趋势比单题 AC 数更重要。

真正的暑假目标不是“又做了多少题”，而是让学生在考场上形成稳定动作：

```text
先保基础分
再冲 T3
最后稳拿 T4 部分分
遇到不会的题也不空题
每个想法都能落到代码和分数上
```
