# 2026 CSP-J 暑期一等奖训练计划（家长版）

## 一、这份计划的目的

这份计划的目标，是让家长清楚了解暑假训练到底在做什么、为什么这样安排、训练到什么程度才有机会冲击 CSP-J 一等奖。

这不是一份“保证获奖承诺书”。竞赛结果会受到学生基础、暑假投入、考试发挥、当年题目难度等因素影响。我们能承诺的是：用清晰的训练路径、可检查的过程、阶段性模拟和及时复盘，把学生已有基础尽量转化为稳定的考试分数。

暑假训练的核心不是简单增加刷题数量，而是解决一个更关键的问题：

```text
能写出暴力解，但还不能稳定把暴力思路升级成正解。
```

如果这个问题解决得好，学生的一等奖竞争力会明显提高。

## 二、学生当前的基本情况

学生已经完成了 `CSP-J 400 分题单` 中的大部分训练，说明基础题量已经有积累，不是从零开始。

目前更需要关注的不是“还要刷多少题”，而是：

1. 能拿到基础分，但还不能稳定突破到高分。
2. 遇到 T3/T4 时，能写出部分暴力解，但经常卡在优化。
3. 有时能想到方向，但无法说明为什么这个做法正确。
4. 知道正解思路后，代码实现和调试还不够稳定。

换句话说，学生现在最需要训练的是从“会做一部分”到“能在考试中稳定拿分”的转化能力。

## 三、一等奖需要达到什么水平

CSP-J 一等奖通常不是靠最后一题完全做出来，而是靠前面题目足够稳定，再加上后面题目的有效得分。

我们的目标分数结构是：

```text
T1：100 分，必须稳定
T2：90-100 分，基本不能丢
T3：70-100 分，是暑假训练的核心突破点
T4：30-60 分，不要求完全做出，但要稳定拿部分分
总分目标：290-330+
```

也就是说，稳一等奖的关键是：

```text
T1/T2 不崩
T3 尽量打穿
T4 不空题，稳定拿暴力分、特殊性质分、部分正解分
```

如果学生能在后期模拟赛中稳定达到这个分数结构，一等奖希望会明显提高。

## 四、暑假 8 周训练安排

暑假训练按 8 周设计，每天建议投入 3-4 小时。

整体分为四个阶段：

```text
第 1-2 周：诊断与 DP 基础转化
第 3-4 周：数学、贪心与规律专项
第 5-6 周：近三年 CSP-J 风格专项
第 7-8 周：模拟赛与稳分策略
```

### 第 1-2 周：诊断与 DP 基础转化

这个阶段的目标是让学生明白：暴力解为什么慢，怎样把暴力中的重复计算变成 DP 状态。

重点训练：

1. 状态如何定义。
2. 转移方程从哪里来。
3. 初始化和边界如何处理。
4. 如何从暴力枚举中抽出可优化的结构。

这阶段不追求题量，而追求每道题都能讲清楚“为什么这样做”。

### 第 3-4 周：数学、贪心与规律专项

这个阶段解决学生常见的两个问题：

```text
能写暴力，但不知道怎么优化。
想到贪心或规律，但说不清为什么正确。
```

训练重点包括：

1. 从小数据和样例中找规律。
2. 根据数据范围反推算法复杂度。
3. 用排序、前缀、二分、局部最优等方法优化暴力。
4. 用简单清楚的语言说明正确性。

从第 3 周开始，每周安排 1 次完整模拟赛，用真实分数检验训练效果。

### 第 5-6 周：近三年 CSP-J 风格专项

这个阶段会集中训练近三年 CSP-J 的题目风格。

重点不是单纯做真题，而是让学生掌握：

1. T3 如何从暴力过渡到正解。
2. T4 如何先拿部分分。
3. 特殊性质分如何识别。
4. 考场上不会完整正解时，如何尽量把会的部分写出来。

近三年真题会做两遍：

```text
第一遍：限时做，记录真实得分。
第二遍：复盘后重写，补齐暴力、部分分和正解。
```

### 第 7-8 周：模拟赛与稳分策略

最后两周不再大规模扩题，而是训练考试转化。

每周安排：

```text
2 次完整模拟赛
2 次错题重写
1 次 T3/T4 专项
1 次 T1/T2 零失误训练
1 天轻量复盘
```

这阶段重点看分数稳定性，而不是看又做了多少新题。

## 五、每天具体怎么训练

每天训练不以“做了几道题”为核心，而以是否完成完整训练闭环为核心。

每日建议流程：

```text
60-90 分钟：限时做 1 道核心题
30 分钟：补全暴力解，确认基础分能拿到
60 分钟：从暴力升级到正解
30 分钟：重写代码、调试或对拍
10 分钟：记录复盘
```

每道核心题都要留下四个问题的答案：

```text
暴力怎么写？
瓶颈在哪里？
正解消掉了哪一层复杂度？
下次看到什么特征要想到这个方法？
```

这四个问题非常重要。它们能判断学生是在真正理解，还是只是在看题解、抄代码。

## 六、如何判断训练是否有效

训练是否有效，不只看“今天做了几题”，而要看下面这些指标。

### 1. T1/T2 是否稳定

T1/T2 是一等奖的底盘。如果 T1/T2 还经常因为读题、边界、变量、数组、输出格式而丢分，说明基础稳定性还不够。

目标是：

```text
T1 基本满分
T2 接近满分
低级错误当天复盘、当天重写
```

### 2. T3 是否能从暴力升级

T3 是一等奖的核心突破点。

我们希望学生逐步达到：

```text
先能写出暴力
再能指出暴力瓶颈
再能想到优化方向
最后能在规定时间内写出正解或高分做法
```

如果 T3 长期只能停留在 30-60 分，说明需要继续加强 DP、贪心、数学规律和复杂度分析。

### 3. T4 是否不空题

T4 不要求每次完整做出，但不能空题。

目标是：

```text
能写暴力分
能识别特殊性质
能拿到 30-60 分
不会因为想不到正解就放弃整题
```

### 4. 模拟赛分数是否稳定上升

第 3 周后开始安排模拟赛。每次模拟赛后要记录：

```text
每题得分
失分原因
低级错误数量
T3/T4 是否写出暴力
是否有时间分配问题
```

后期如果能稳定达到 290 分以上，说明一等奖竞争力已经比较明确。

## 七、可能出现的问题

### 问题 1：看起来每天做题不多

这是正常的。

本阶段不是低年级入门刷题，而是一等奖冲刺训练。每道核心题都要经历暴力、优化、证明、代码、复盘。做一题真正吃透，比一天刷很多题但没有复盘更有价值。

### 问题 2：模拟赛分数会波动

这也是正常的。

模拟赛的目的不是每次都考高分，而是暴露问题。只要每次赛后能明确知道丢分原因，并在下一次训练中修正，波动就是有价值的。

真正需要警惕的是：

```text
每次都犯同类错误
T1/T2 反复低级失误
T3 长期只会暴力
T4 长期空题
复盘只写“不会”，没有具体原因
```

### 问题 3：学生觉得自己会了，但考试写不出来

这说明还没有达到“考试可用”的程度。

真正过关不是看懂题解，而是：

```text
能在限时内独立写出
能解释关键转移或贪心理由
能处理边界和特殊情况
能通过样例和自测
```

所以暑假后半段会加大模拟赛和重写训练。

### 问题 4：家长担心一等奖不确定

这个担心是合理的。竞赛本来就有不确定性。

我们会把不确定性转化为可观察的过程指标：

```text
T1/T2 是否稳定
T3 是否能突破 70+
T4 是否能稳定拿部分分
模拟赛是否逐步接近 290-330+
错题是否能重写成功
```

如果这些指标逐步达成，一等奖希望会明显提高。

## 八、家长需要如何配合

家长最重要的配合，不是每天追问“今天做了几道题”，而是帮助学生保持稳定节奏，并关注训练质量。

建议家长关注这些问题：

```text
今天有没有完成复盘？
这道题是只看懂了，还是能独立重写？
这周 T1/T2 有没有低级错误？
T3 有没有从暴力升级到更高分做法？
模拟赛丢分原因有没有记录清楚？
```

不建议的做法：

```text
只用做题数量评价训练效果
一次模拟赛低分就否定整个训练
一次模拟赛高分就认为已经稳了
频繁打断训练节奏，临时更换目标
只关注排名，不关注具体失分原因
```

家长真正需要帮助学生做到的是：

```text
每天有固定训练时间: 每天3-4小时!!!!!
每周完成模拟和复盘
低级错误及时清零
遇到难题不逃避，也不死磕过久
训练过程持续 8 周不断档
```

## 九、成功可能性与风险判断

如果学生能坚持 8 周训练，完成核心题复盘，并在后期模拟赛中稳定达到 290 分以上，一等奖希望较大。

如果后期能稳定达到：

```text
T1：基本满分
T2：接近满分
T3：70 分以上，部分场次能接近满分
T4：稳定 30-60 分
```

那么这就是比较健康的一等奖竞争状态。

风险较高的情况包括：

```text
T1/T2 仍然频繁低级失误
T3 长期只能写暴力，无法突破 60 分
T4 长期空题
看题解很多，但独立重写很少
模拟赛后不复盘，错题反复错
暑假训练断断续续，无法形成连续积累
```

如果出现这些情况，就需要及时调整训练重点，优先补稳定性和考试转化，而不是继续盲目加题。

## 十、最终希望达到的状态

暑假结束时，我们希望学生形成稳定的考试动作：

```text
先保 T1/T2
再集中突破 T3
最后争取 T4 部分分
不会的题先写暴力
想不到正解时先拿能拿的分
做完后会检查，赛后会复盘
```

真正接近一等奖的学生，不一定每道难题都能完整做出，但必须具备稳定得分能力。

这份暑假计划的核心，就是把学生从“会做一些题”训练到“能在考试中稳定拿分”。
